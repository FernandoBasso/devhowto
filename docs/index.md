# Dev How To

## Intro

This site contains tutorials, guides, examples, tips, notes, complaints,
rantings and many other things related to using computers and software
development. It is a compilation of my own work mixed with ideas and
notes I learn from others. You'll find pages with tutorial-like
explanations on things that interest me and code solutions for book
exercises, challenge websites, etc.
